## 1. 核心视觉进一步升华：意境排版 (Zen Typography & Layout)
在“水墨宣纸”的基础上，从“形”转向“意”，通过极致的留白和排版，实现“素雅、美观、大方”。

## 2. 细化实施方案

### **第一阶段：极致文字布局 (Type.kt & Typography Refinement)**
- **呼吸感排版**：
  - 增加 `letterSpacing`（字间距）至 `0.8.sp` - `1.2.sp`，模拟古籍拓片的疏朗感。
  - 将 `lineHeight`（行高）进一步提升，正文行高设定为字号的 1.7 - 1.8 倍，确保长文阅读不压抑。
- **层级清晰化**：
  - 定义完整的 `Typography` 体系，包括 `titleLarge`（章节感）、`bodyLarge`（正文流）、`labelSmall`（旁注感）。
  - 关键标题采用 `FontWeight.SemiBold` 配合较大的字间距，展现“大方”气度。

### **第二阶段：去组件化布局 (Minimalist Chat Layout)**
- **气泡形态进化**：
  - **AI 消息**：彻底移除背景色，采用“无界”设计。仅在左侧保留一根极细的“墨线” (Ink Line) 作为指引，使内容直接浮现在宣纸背景上，仿佛直接书写在纸上。
  - **用户消息**：改为极淡的“烟灰”色块，边缘不再使用圆润的 20dp，而是改为微小的 `4.dp` 圆角，模拟信笺的方正。
- **间距疏朗化**：
  - 将消息条目间的垂直间距从 `24.dp` 增加到 `32.dp`，增加视觉冗余度，体现“素雅”。

### **第三阶段：交互细节的“墨韵”处理 (Polishing UI Elements)**
- **分割线艺术**：移除所有实线分割，改为渐变消失的“墨迹线”或直接利用留白分割。
- **对话框精修**：
  - `AlertDialog` 容器改为更加方正的线条，背景使用宣纸质感。
  - 按钮改为“纯文字”或“边框式”设计，取消实色填充，减少视觉侵略性。
- **图标微调**：所有图标缩小 10%，并降低透明度至 0.6，使其不喧宾夺主。

### **第四阶段：文案雅化 (Refined Language)**
- 升级全局 UI 文案：
  - “搜索会话” -> “寻迹”
  - “模型与 Key 配置” -> “墨池配置” 或 “灵犀设定”
  - “重命名” -> “题名”
  - “删除” -> “抹除”
  - “发送” -> “投递”

## 3. 质量保障
- **视觉一致性**：确保深色模式下依然保持“玄色纸张”的通透感，文字颜色采用“月白”而非纯白。
- **性能优化**：对宣纸纹理进行缓存处理，确保滑动流畅。
- **编译验证**：全量代码检查，确保无逻辑错误。
